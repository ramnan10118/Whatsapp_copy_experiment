## The 3-Second Skim Test
Before finalising any message:
1. Is the main benefit visible in the first line?
2. Can you understand the whole offer in 3 seconds?
3. Does it sound like a human texted this, not a company?
4. Is the CTA exactly what the user will see after tapping?
